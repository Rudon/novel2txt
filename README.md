## Welcome to novel2txt  ##
开源PHP在线小说txt生成器，欢迎使用！



### How to install it? ###
* 打开https://github.com/RudonTeam/novel2txt
* Clone or Download > Download Zip > 解压到服务器项目目录下，重命名目录为`novel2txt`
* 完成
* 打开https://your-site.com/novel2txt 即可享用
